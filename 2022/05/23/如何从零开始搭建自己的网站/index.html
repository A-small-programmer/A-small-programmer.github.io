    前言

    此文章将讲述如何搭建网站，并使用我的主题，当然用这位大佬云游君的也行哦!

    步骤

    准备工作

    安装node.js

    注意，安装长期支持版，而不是当前发布版!

    安装git和vscode

    git到官网下载，至于vscode就不用我提了吧。

    注册github

    在github上注册一个账号就行了。
    在主页上点击new -> new repository键入仓库名你的用户名+github.io

    安装Hexo

    在终端输入以下命令:

    npm install hexo-cli -g

    此时，请先通过cd进入你本地电脑打算存储网站代码的文件夹目录。
    接下来输入:

    hexo init 你的名字.github.io
cd 你的名字.github.io
npm install
hexo server

    如不行请加npx

    使用Hexo主题
    下载我的主题

    输入:

    git clone https://github.com/A-small-programmer/hexo-theme-dipt

    这里顺带求个star
    将文件夹移动进node_modules

    配置Hexo文件

    把_config.yml中的theme字段的landscape改为dipt
    再把language字段改为zh-CN
    然后把author字段改为你的昵称

    配置主题

    进入node_modules文件夹，在进入hexo-theme-dipt文件夹，找到里面的_config.yml
    输入:

    title: 站点标题

author: 你的昵称

menus: 
  主页: /
  关于: /about/site

links:
  dipt v0.0.1: https://github.com/A-small-programmer/hexo-theme-dipt
  github: 你的github网址
    生成静态文件

    由于github pages只支持纯静态文件，所以我们要生成静态文件，输入:

    npx hexo generate

    此时你的文件夹目录下会出现public这个文件夹，里面存放的就是你站点的静态文件。

    部署

    下载github desktop，clone你的仓库，将文件移动进文档/Github/仓库名里，点蓝色按钮，点push，OK!

    写作

    在source/_posts里面新建.md文件，开始写作吧!
